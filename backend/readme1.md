# Backend Setup Guide

Bu dosya otomatik olarak güncellendi. Lütfen `README.md` dosyasına bakın.

## Hızlı Başlangıç

1. Dependencies'i yükle: `npm install`
2. `.env` dosyasını oluştur: `cp .env.example .env`
3. Database migration'ları çalıştır: `npm run migrate`
4. Sunucuyu başlat: `npm run dev`

Detaylı bilgi için `README.md` dosyasına bakın.
